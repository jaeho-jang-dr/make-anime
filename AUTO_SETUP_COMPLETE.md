### 2️⃣ Whisk에서 영상 생성 (이것만 수동!)

**이유**: Whisk는 웹 인터페이스라 자동화 불가능

**Whisk 바로 가기**: https://labs.google/fx/tools/whisk

**가장 간단한 테스트 (5분)**:

#### 입력 3가지:

1. **Subject**: 혜완이 사진 업로드
   ```
   파일: local_files/anime-pipeline/characters/hyewan_original.jpg
   ```

2. **Scene**: 프롬프트 입력
   ```
   Korean house entrance, warm lighting
   ```

3. **Style**: 프롬프트 입력
   ```
   warm family documentary, photorealistic
   ```

#### 생성:
1. **Create** 클릭 (이미지 생성)
2. **ANIMATE** 클릭 (애니메이션)
3. 프롬프트 입력:
   ```
   child running into house, natural motion
   ```
4. **Generate** 클릭
5. 다운로드: `scene_006_test.mp4`
