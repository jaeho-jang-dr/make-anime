## 클릭 한 번씩 따라하기

---

## 📱 시작 전 준비

### ✅ 필요한 것:
```
1. Whisk 사이트: https://labs.google/fx/tools/whisk (이미 열려있음 ✅)
2. 혜완이 사진: local_files/anime-pipeline/characters/hyewan_original.jpg
3. 이 가이드
```

---

## 🎯 전체 흐름 (3단계)

```
1단계: 이미지 만들기 (Create)
   ↓
2단계: 애니메이션 만들기 (Animate)
   ↓
3단계: 다운로드
```

**소요 시간**: 약 5-10분

---

## 📋 1단계: 이미지 만들기

### Step 1-1: Subject (주인공) 설정

**화면 왼쪽에 "Subject" 영역이 있습니다**

```
클릭할 곳: "Upload image" 또는 "+" 버튼
```

**그 다음**:
1. 파일 선택 창이 열림
2. 이동: `D:\Entertainments\DevEnvironment\make-anime\local_files\anime-pipeline\characters`
3. 파일 선택: `hyewan_original.jpg`
4. "열기" 클릭

**결과**: 혜완이 사진이 왼쪽 박스에 나타남 ✅

---

### Step 1-2: Scene (배경) 설정

**화면 중앙에 "Scene" 영역이 있습니다**

```
방법 1: 텍스트 입력 (추천)
```

**클릭할 곳**: Scene 아래의 **텍스트 입력창**

**붙여넣기** (Ctrl+V):
```
Korean house entrance, warm lighting, natural atmosphere
```

또는

**방법 2: 밀양 집 사진 업로드**:
```
"Upload image" 클릭
→ local_files/01-외부_01[1].jpg 선택
```

---

### Step 1-3: Style (스타일) 설정

**화면 오른쪽에 "Style" 영역이 있습니다**

**클릭할 곳**: Style 아래의 **텍스트 입력창**

**붙여넣기** (Ctrl+V):
```
warm family documentary, photorealistic, natural lighting
```

---

### Step 1-4: Create 버튼 클릭!

**화면 아래쪽에 큰 버튼이 있습니다**

```
클릭할 곳: "Create" 또는 "Mix" 버튼 (노란색 또는 파란색)
```

**결과**:
- 로딩 시작 (1-2분)
- 여러 개 이미지가 생성됨
- 4-6개 정도의 이미지 그리드

---

## 📋 2단계: 애니메이션 만들기

### Step 2-1: 이미지 선택

**생성된 이미지들 중 하나를 선택합니다**

```
클릭할 곳: 마음에 드는 이미지 1개 클릭
```

**선택 기준**:
- 혜완이가 잘 나온 것
- 배경이 자연스러운 것
- 전체적으로 깔끔한 것

---

### Step 2-2: ANIMATE 버튼 찾기

**이미지를 선택하면 나타나는 버튼들**

```
찾을 버튼: "ANIMATE" (보통 이미지 아래 또는 옆)
```

**위치**:
- 선택한 이미지 바로 아래
- 또는 이미지 확대 화면에서 하단
- "Remix" 버튼 옆에 있을 수 있음

```
클릭할 곳: "ANIMATE" 버튼
```

---

### Step 2-3: 애니메이션 프롬프트 입력

**ANIMATE 클릭하면 새 화면이 나타납니다**

```
보이는 것: 애니메이션 설명 입력창
```

**클릭할 곳**: 텍스트 입력창

**붙여넣기** (Ctrl+V):
```
child running into house, natural movement, joyful
```

또는 더 간단하게:
```
natural walking motion
```

---

### Step 2-4: Generate 버튼 클릭!

**프롬프트 입력 후**

```
클릭할 곳: "Generate" 또는 "Create animation" 버튼
```

**결과**:
- 로딩 시작 (2-5분) ⏳
- 진행 상태 표시
- 완료되면 8초 영상 재생!

---

## 📋 3단계: 다운로드

### Step 3-1: 영상 확인

**생성된 영상이 자동 재생됩니다**

```
확인할 것:
- 움직임이 자연스러운가?
- 8초 길이인가?
- 만족스러운가?
```

---

### Step 3-2: 다운로드 버튼 클릭

**영상 플레이어 주변을 보세요**

```
찾을 버튼: "Download" 또는 다운로드 아이콘 (↓)
```

**위치**:
- 영상 플레이어 아래
- 또는 오른쪽 상단
- 점 3개(...) 메뉴 안에 있을 수 있음

```
클릭할 곳: "Download" 버튼
```

---

### Step 3-3: 파일 저장

**파일 저장 창이 나타납니다**

```
저장 위치: D:\Entertainments\DevEnvironment\make-anime\local_files\anime-pipeline\scenes\clips
파일명: scene_006_test.mp4
```

**클릭**: "저장" 버튼

---

## ✅ 완료!

**축하합니다! 첫 영상을 만들었습니다!** 🎉

## 🔄 다음 장면 만들기

**같은 과정 반복**:
```
1. Subject: 혜완이 사진 (또는 할아버지, 할머니)
2. Scene: 다른 배경 프롬프트
3. Style: 같은 스타일
4. Create → 선택 → ANIMATE → Generate
5. 다운로드
```

## 💡 꿀팁

### 더 쉬운 방법:
```
1. Scene과 Style은 같은 것 계속 사용해도 됨
2. Subject만 바꿔가며 여러 장면 제작
3. 애니메이션 프롬프트만 조금씩 변경
```

### 시간 절약:
```
- 프롬프트 미리 복사해두기
- 여러 탭 열어서 동시에 여러 개 만들기
- 마음에 안 들면 "Remix" 클릭해서 다시 생성
```

## ❓ 버튼을 못 찾겠어요?

### ANIMATE 버튼이 안 보여요:
```
해결책:
1. 이미지를 한 번 더 클릭
2. 이미지 확대 화면에서 찾기
3. 화면 스크롤 내려보기
4. 점 3개(...) 메뉴 열어보기
```

### Create 버튼이 안 보여요:
```
해결책:
1. 화면 아래쪽 확인
2. 스크롤 내리기
3. Subject, Scene, Style이 모두 입력되었는지 확인
```
