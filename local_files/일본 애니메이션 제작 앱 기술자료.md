### **📚 4\. 2D 애니메이션 기술 자료**

#### **애니메이션 12원칙 (Disney Studios 정립)**

1. Squash & Stretch (압축과 늘림)  
2. Anticipation (예비 동작)  
3. Staging (연출)  
4. Straight Ahead / Pose to Pose  
5. Follow Through & Overlapping Action  
6. Slow In & Slow Out (이징)  
7. Arcs (호 궤적)  
8. Secondary Action (부수 동작)  
9. Timing (타이밍)  
10. Exaggeration (과장)  
11. Solid Drawing (입체적 드로잉)  
12. Appeal (매력)

#### **프레임 레이트 기준**

* 일본 애니: 보통 24fps, 2-3프레임에 1장 그림 (on twos, on threes)  
* TV용: PAL 25fps / NTSC 30fps

## **1\. OpenToonz ⭐ 추천**

* **설명**: 스튜디오 지브리가 실제로 사용하는 2D 애니메이션 소프트웨어  
* **장점**:  
  * 완전 무료 오픈소스 (상업적 사용 가능)  
  * GTS 스캐닝 도구 포함  
  * 머신러닝/딥러닝 효과 플러그인 지원  
* **GitHub**: https://github.com/opentoonz/opentoonz  
* **웹사이트**: https://opentoonz.github.io/e/

## **2\. Anime.js ⭐ AI 통합에 최적**

* **설명**: 경량 JavaScript 애니메이션 라이브러리  
* **장점**:  
  * 매우 가벼움 (3-10KB)  
  * 강력한 API로 프로그래밍 방식 제어 가능  
  * CSS, SVG, DOM 속성 등 모든 웹 요소 애니메이션 가능  
  * **AI로 생성한 콘텐츠를 애니메이션화하기 완벽**  
* **설치**: `npm i animejs`  
* **GitHub**: https://github.com/juliangarnier/anime  
* **문서**: https://animejs.com/documentation/

// 사용 예시  
import { animate } from 'animejs';

animate('.character', {  
  x: 320,  
  rotate: { from: \-180 },  
  duration: 1250,  
  ease: 'inOutQuint',  
  loop: true  
});

## **3\. AniSora (Index-AniSora)**

* **설명**: Bilibili가 개발한 오픈소스 애니메이션 비디오 생성 AI 모델  
* **장점**:  
  * 애니메이션 스타일 전문화  
  * 이미지→비디오 생성  
  * 프레임 보간 기능  
* **GitHub**: Bilibili의 Index-AniSora 프로젝트  
* **모델**: CogVideoX-5B, Wan2.1-14B 기반

## **4\. Synfig Studio**

* **설명**: 무료 오픈소스 2D 애니메이션 소프트웨어  
* **장점**:  
  * 벡터 기반 애니메이션  
  * 프레임 단위 작업 불필요  
  * 다른 그래픽 소프트웨어와 통합 가능  
* **웹사이트**: https://www.synfig.org/

## **5\. Blender**

* **설명**: 완전 무료 오픈소스 3D 애니메이션 툴  
* **장점**:  
  * 2D/3D 모두 지원  
  * 대형 스튜디오에서도 사용  
  * Python API로 자동화 가능  
* **웹사이트**: https://www.blender.org/
